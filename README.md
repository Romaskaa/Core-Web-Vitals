Анализ Core Web Vitals
Входные параметры: URL адрес сайта, который нужно проанализировать
Система использует Google PSI API для технического SEO
Возвращается в виде деатльного ответа, который понятен пользователю не погруженному в SEO (AI синтезирует данные в понятный ответ)
Результат анализа можно визуализировать в любом удобном для пользователя формате (различные диаграммы, инфографика, схемы, mermaid)
